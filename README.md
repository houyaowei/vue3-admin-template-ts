# vue3-admin-template-ts
vue3+ ts skeleton，


## 注意：目前是alpha版本

## 开发建议
1、在views组件中定义各个模块，router为标准路由配置，会自动识别并注册

2、环境文件自己添加，并在增加scripts中增加配置

3、工具建议从npm和yarn转到pnpm

## 待办项
1、utils文件下的方法需要同步改为ts版

2、类型文件是否需要集中存放待定

3、store中约束需要再加强

4、其他中间件增强
